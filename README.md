# :hatching_chick: 삐약삐약
꾸준히 잘 챙겨 먹어야 하는 약,  
복용을 깜빡 잊고 있진 않나요?  
  
삐약삐약은 꾸준한 약 복용 습관을 형성하기 위해  
약의 복용여부를 기록할 수 있고,  
깜박 잊은 약을 알림으로 확인할 수 있고,  
목표 개수를 설정해 성취감을 고취시킬 수 있는 앱입니다.
